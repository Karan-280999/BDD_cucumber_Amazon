# BDD Cucumber Amazon Automation Framework

## Tech Stack
- Java 17
- Selenium 4.40
- Cucumber 7.34
- JUnit 4.13
- Extent Reports 5.1
- Jackson (JSON)

## Features Covered
- TC01 - Login and Logout
- TC02 - Login to Checkout

## Framework Highlights
- BDD with Gherkin feature files
- JSON driven test data
- Generic data reader using Reflection
- Reusable utility methods
- Extent HTML Reports
- Screenshot on pass and fail
- Tags support for selective execution
- Parallel execution ready

## How to Run
1. Clone the repo
2. Open in IntelliJ
3. Run Runner.java
